if 조건문 :

- 조건이 참이면, 실행

print("조건이 참")

else :

- if 조건이 거짓인 경우 실행

print("조건이 거짓")


```python
a_if = True
if a_if :
    print("a_if는 True입니다.")
else :
    print("너는 거짓이야!")
```

    a_if는 True입니다.



```python
# 한줄로 바꾸는 법
print("a_if는 True입니다.") if a_if else print("너는 거짓이야!")
```

    a_if는 True입니다.
