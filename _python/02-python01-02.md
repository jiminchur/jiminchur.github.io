### 숫자형

#### 정수형
정수형(lnteger)이란 말 그대로 정수를 뜻하는 자료형을 말한다.


```python
a_int = 100
b_int = -100
c_int = 0
```


```python
# type()은 파이썬 변수의 타입을 확인해주는 함수이다.
type(a_int)
```




    int



#### 실수형
파이썬에서 실수형(Floating-point)은 소수점이 포함된 숫자를 말한다.


```python
t_flo = 5.12
Y_flo = 12.0
```


```python
type(t_flo)
```




    float



##### 컴퓨터식 지수 표현방식


```python
a_flo = 12.789e10
a_flo
```




    127890000000.0




```python
b_flo = 78.456e-4
b_flo
```




    0.0078456




```python
(type(a_flo), type(b_flo))
```




    (float, float)



#### 숫자형 연산

##### 사칙연산
- "+" : 더하기
- "-" : 빼기
- "*" : 곱하기
- "/" : 나누기
- "**" : 제곱
- "%" : 나머지만 
- "//" : 몫만


```python
# "+" : 더하기
a_int = 4
b_int = 5
a_int + b_int 
```




    9




```python
# "-" : 빼기
a_int - b_int
```




    -1




```python
# "*" : 곱하기
a_int * b_int
```




    20




```python
# "/" : 나누기
a_int / b_int
```




    0.8




```python
# "**" : 제곱
a_int ** b_int
```




    1024




```python
# "%" : 나머지만
a_int % b_int
```




    4




```python
# "//" : 몫만
a_int // b_int
```




    0
