### 실습하기 (2)

1. 파일 하나 만들기
    
    ![실습01](/assets/images/gcc01.png)
* hello.c 파일 코드
    ```
    #include <stdio.h>

    int main()
    {
            printf("hello world\n");
            return 0;
    }
    ```
2. gcc 이용해서 컴파일 하기
    
    ![실습02](/assets/images/gcc04.png)
    ```
    gcc -o hello hello.c
    ```
3. hello파일 실행 해보기

    ![실습03](/assets/images/gcc03.png)
    ```
    ./hello
    ```
4. 어셈블리어 만들기
    
    ![실습04](/assets/images/gcc05.png)
    ```
    gcc -S hello.c -o hello.S
    ```
5. 파일 확인하기

    ![실습05](/assets/images/gcc06.png)
    ```
    cat hello.S
    ```
